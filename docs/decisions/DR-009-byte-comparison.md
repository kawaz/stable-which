# DR-009: ファイル同一性判定をバイト単位比較に変更

## 決定

`DefaultHasher` によるハッシュ比較を廃止し、ファイルサイズ比較 → バイト単位ストリーミング比較に変更する。

## 理由

- `DefaultHasher` は暗号学的ハッシュではなく 64bit で衝突容易（birthday attack で ~2^32 試行）
- Rust バージョン間でアルゴリズムの安定性が保証されていない
- バイト単位比較は完全な正確性を保証し、依存ゼロを維持できる
- ファイルサイズの事前比較で大半のケースを O(1) で棄却可能

## 不採用案

- SHA-256: 正確だが `sha2` crate の依存が必要（ライブラリの依存ゼロ方針に反する）
- DefaultHasher 維持: セキュリティ用途に不適切
